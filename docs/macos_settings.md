# MacOS Settings
List of tricks for easier usage

## iTerm2

[iTerm2](https://iterm2.com/) is generally more convenient than the default Terminal.

## Shortcuts from Finder into Terminal

System Settings > Keyboard > Keyboard Shortcuts > Services > Files and Folders >

If you right click in a Finder folder, at the bottom of the dropdown will be Services and one of the options will be "New Terminal at Folder"
